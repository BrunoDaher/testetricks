Formulários

Formulários são utilizados pra ler dados da página e enviar, ou
mesmo executar cálculos.

Elementos de formulário e seletores Form - Formulário para entrada de dados
TextField - Campo para texto
TextArea - Campo para texto grande
Button - Botão de ação
Label - Rótulo para inputs
Input - Campo para entrada de dados
Inputs São varios os tipos de Input, entre eles:
  • type="number" - formato adequado para números
    Idade
  • type="search" - ideal para pesquisas - pode conter dados de uma lista
    Pesquisa
  • type="password" - ideal para pesquisas - pode conter dados de uma lista
    Idade
  • type="url" - aceita somente caminhos url
    Url
  • type="range" - uma barra deslizante
    Volume
Uma observarção interessante: o label é mais que um mero rótulo para um input, sob ação do mouse ele direciona o cursor para o campo à qual faz referência.
Uma lista completa de ações pode ser encontrada em https://developer.mozilla.org/.
Seletores Seletores são chaves que identificam um elemento HTML, como por exemplo o "id" ou a "class" (classe). Através dos seletores é possível atribuir ações JavaScript aos elementos e determinar o comportamento deles conforme as ações do mouse, e outros.
  1. Id - representação "#"
  2. class - representação - "."
Exemplo
<input class='customInput' id='nome'>

No código o seletor id recebe o valor "nome", e o seletor class recebe o valor "customInput". o JavaScript pode atribuir uma função específica a este input, ou mesmo ler os valores nele contido através do seletor id. O navegador lê o código HTML e o CSS, e se utiliza dos seletores class ou mesmo do id no momento de renderizar a página. Caso haja alguma referência ao elemento HTML nó código CSS, desde que corretamente escrito, o navegador processa e escreve respeitando as regras definidas, como regras de cor, fonte e outras mais.

Em código JavaScript var x = document.getElementById('nome');

O código acima atribui à variavel X o elemento input , cujo id é "nome". A partir disso é possível realizar operações diversas.

Em código Css .customInput {color:blue;font-size:10px;}

O código acima atribui à todo elemento que pertencer à "customInput" as características escritas dentro das chaves, como color e font-size, que são atributos de css.
