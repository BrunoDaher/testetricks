Firebase Firebase é uma ferramenta do google cloud e é útil para hospedar um código assim como fazer funções de backend(autenticação de usuário, estatísticas e etc) e base de dados. Para instalar o firebase é necessário seguir uma sequência de passos, descritos no próprio site da google. Tendo uma conta google você já tem o que precisa para acessar o firebase.

Instalação É necessário fazer a instalação do Node.js para poder instalar as ferramentas do firebase no console.

Na sequencia instalar NVM (gerenciador do Node), e em seguida o Firebase.
Passo a passo retirado de https://firebase.google.com/docs/cli

Após a instalação do node (apenas next...next...next no instalador), se tudo tiver devidamente instalado, basta digitar no terminal o seguinte comando:
npm install --save firebase

npm install --save firebase-tools
