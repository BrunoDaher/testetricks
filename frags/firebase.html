DOM (Document Object Model) O DOM é a interface que permite leitura e alteração de documentos HTML, SVG e XML, linguagens de estrutura e anotação por nós ou aninhamento. Entre outras linguagens, o JavaScript lê esta estrutura aninhada e através do DOM (uso de api) pode controlar o comportamento dos elementos da página, configurações CSS, bem como gerenciais configurações do navegador para a página em questão, como memória de sessão e memória local. Com controle de eventos e gerencia de animações é possível até mesmo fazer jogos em JS, ainda que pequenos. Mas já seguem em testes jogos FPS, através de uso de APIs mais apropriadas pra renderização 3D.
Seletores Através dos seletores o JS pode alterar ou atribuir eventos aos elementos da página, tais como:
  • name
  • id
  • class
  • tags - ('body', 'footer', 'main' ...etc
Os seletores podem ser atributos ou mesmo a própria tag em que estao encapsulados os elementos, ou mesmo, aninhados.
