Javascript Javascript é uma linguagem de programação não tipada que admite o paradigma de orientação à objetos. Diferente de C(c++, c#), Java e demais, não se trata de uma linguagem tipada. O JavaScript é poderoso e consegue criar todo tipo de aplicação para ser consumida através dos browsers e atualmente em apps mobile ou mesmo desktop. 
JavaScript
    var x = 10;
    x =  'ola';
    x = new Object();

Java
    Int x = 10;
    String x = 'olq';
    Object x = new Object();
O exemplo acima demonstra que em linguagens a linguagem tipada (Java) e a não tipada (JavaScript). X , no exemplo acima, vale 10, logo em seguida ele recebe o valor "olá", passou de number para string. Isso seria impossível em linguagem Java, por exemplo, sem uma transformação - parse. Isto é, teria-se que transformar um número em string, ou uma string em número, através de funções.

As variáveis no JavaScript pode serm : Globais (var) Locais (Const e Let) Imutáveis (Const) - não permite a troca de tipagem, como as anteriores.
Se o x do exemplo acima fosse uma const

DOM (Document Object Model) O DOM é a interface que permite leitura e alteração de documentos HTML, SVG e XML, linguagens de estrutura e anotação por nós ou aninhamento. Entre outras linguagens, o JavaScript lê esta estrutura aninhada e através do DOM (uso de api) pode controlar o comportamento dos elementos da página, configurações CSS, bem como gerenciais configurações do navegador para a página em questão, como memória de sessão e memória local. Com controle de eventos e gerencia de animações é possível até mesmo fazer jogos em JS, ainda que pequenos. Mas já seguem em testes jogos FPS, através de uso de APIs mais apropriadas pra renderização 3D.
Seletores Através dos seletores o JS pode alterar ou atribuir eventos aos elementos da página, tais como:
  • name
  • id
  • class
  • tags - ('body', 'footer', 'main' ...etc
Os seletores podem ser atributos ou mesmo a própria tag em que estao encapsulados os elementos, ou mesmo, aninhados.
