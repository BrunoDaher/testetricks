Navegação Web:

Um site é visitado normalmente através do navegador (chrome, mozilla, safari, ie), pc ou mobile. O cliente, ao acessá-los faz uso do frontend, enquanto o back-end corresponde à programação que fica no lado do servidor.

Em outras palavras, frontend é o código do lado do cliente, backend fica do lado do servidor.

Linguagens Front End
HTML, CSS, JavaScript
Backend
Java, PHP, JavaScript(Node.js), Python.

HTML O HTML (HyperText Markup Language) é uma linguagem de marcação/anotação. A estrutura do site e os elementos são "anotados" no documento html Um documento HTML é um documento de anotações, onde os elementos são representados pelas tags, e com a técnica de aninhamento estabelece-se a hierarquia dos mesmos. Um navegador de interent é, entre outras coisas, um interpretador de textos, entre eles o HTML.

CSS O CSS Cascate Style Sheet é a linguagem onde são definidos os estilos visuais dos elementos HTML, tais como:
  • Cores, Filtros, Bordas
  • Plano de fundo
  • Estilos de cursor de mouse conforme eventos.
  • Animação
  • Personalização proporcional ao dispositivo
 Tanto HTML como CSS podem ter comportamentos distintos conforme o navegador usado, tais como (Chrome, Firefox, Edge, I.E., Safari, Opera e etc.);

JavaScript O JavaScript é de fato uma linguagem de programação, orientada à objetos, apesar de ser também considerada uma linguagem orientada à eventos. JavaScript permite:
  • Alterações de estilos visuais
  • Controle de Eventos
  • Gestão de Sessão
  • Gestão de acesso à Banco de Dados
  • Requisições Http - Rest
  • Chamadas REST
  • Construção de elementos gráficos - canvas
